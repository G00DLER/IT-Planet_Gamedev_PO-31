# IT_Planet

### RPG-игра в стиле Souls-like.
### Данная игра была выполнена к соревнованиям от Braim, партнерой которой является компания SberTech.
### Главная задача игры: истребить всех врагов, выжить различными способами: увороты, прыжки.
### Игра имеет поддержку геймпада.
